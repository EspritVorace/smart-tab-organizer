## Configuration générale

<ThemeImage lightSrc={settingsMiscLight} darkSrc={settingsMiscDark} alt="Page des paramètres généraux" />

La page des paramètres regroupe les options de comportement de l'extension :

- **Groupement automatique** — active ou désactive le groupement des onglets lors de leur ouverture
- **Déduplication** — configure le mode de détection des onglets en doublon (URL exacte, chemin, domaine, ou sous-chaîne)
- **Notifications** — active les notifications lors d'un groupement ou d'une déduplication
- **Intervalle de synchronisation** — durée entre deux synchronisations automatiques des profils épinglés
