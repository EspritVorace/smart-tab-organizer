Sessions let you capture a snapshot of your open tabs and restore them later, in full or in part.
Pinned profiles go further: they sync automatically and can be tied to a dedicated window.

## Sessions list

<ThemeImage lightSrc={sessionsListLight} darkSrc={sessionsListDark} alt="List of saved sessions with pinned profiles" />

Sessions are displayed in chronological order. Pinned profiles appear at the top with their distinctive icon.

## Create a snapshot

<ThemeImage lightSrc={sessionsCreateLight} darkSrc={sessionsCreateDark} alt="Snapshot creation wizard" />

Click "New snapshot" to capture the tabs of the active window.
You can choose a name, add a note, and select which tabs to include.

## Notes

<ThemeImage lightSrc={sessionsNoteLight} darkSrc={sessionsNoteDark} alt="Session card with expanded note and search highlighting" />

Each session can include a free-text note. Note content is included in global search.

## Pinned profiles

<ThemeImage lightSrc={sessionsProfilePinLight} darkSrc={sessionsProfilePinDark} alt="Onboarding wizard when pinning the first profile" />

Pin a session to make it a permanent profile.
A profile can be linked to a browser window: the extension automatically syncs that window's tabs with the profile at the configured interval.

## Search

<ThemeImage lightSrc={sessionsSearchLight} darkSrc={sessionsSearchDark} alt="Deep search with filtered session previews" />

Search spans session names, notes, tab titles, and saved URLs.
Results show a preview of matching tabs with highlights.

## Restore with conflicts

<ThemeImage lightSrc={sessionsRestoreConflictLight} darkSrc={sessionsRestoreConflictDark} alt="Restore wizard showing conflict resolution" />

When restoring, if tabs are already open, the extension detects conflicts and lets you skip, replace, or open them as duplicates.

## Import / Export

### Export sessions

<ThemeImage lightSrc={sessionsExportLight} darkSrc={sessionsExportDark} alt="Export dialog with split button" />

Export one or more sessions as JSON for backup or transfer.

### Import sessions

<ThemeImage lightSrc={sessionsImportFileLight} darkSrc={sessionsImportFileDark} alt="Import sessions from file dialog" />

Import sessions from a JSON file or by pasting text.

<ThemeImage lightSrc={sessionsImportConflictsLight} darkSrc={sessionsImportConflictsDark} alt="Conflict resolution when importing sessions" />

In case of name conflicts, choose to rename, overwrite, or skip duplicate sessions.
