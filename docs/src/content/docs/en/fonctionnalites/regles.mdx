Domain rules automatically group tabs based on their domain or a regex pattern.
They apply when opening a tab via middle-click or right-click → "Open in new tab".

## Rules list

<ThemeImage lightSrc={rulesListLight} darkSrc={rulesListDark} alt="Domain rules list with badges and toggles" />

Each rule shows its domain, matching mode, and state (enabled/disabled).
Available actions include editing, deleting, and reordering via drag-and-drop.

## Create a rule

Rules are created through a 4-step wizard.

### Step 1 — Identity

<ThemeImage lightSrc={rulesCreateStep1Light} darkSrc={rulesCreateStep1Dark} alt="Step 1: enter domain and group name" />

Enter the target domain and the group name that will be created in Chrome/Firefox.

### Step 2 — Matching mode

Three modes are available:

**Preset** — uses a predefined regex pattern from a built-in library.

<ThemeImage lightSrc={rulesCreatePresetLight} darkSrc={rulesCreatePresetDark} alt="Preset mode with predefined pattern selection" />

**Ask** — prompts for confirmation each time before grouping.

<ThemeImage lightSrc={rulesCreateAskLight} darkSrc={rulesCreateAskDark} alt="Ask mode: confirmation on each tab open" />

**Manual** — you enter your own regular expression.

<ThemeImage lightSrc={rulesCreateManualLight} darkSrc={rulesCreateManualDark} alt="Manual mode: custom regex input" />

### Step 4 — Summary

<ThemeImage lightSrc={rulesCreateSummaryLight} darkSrc={rulesCreateSummaryDark} alt="Rule summary before creation" />

Review the settings before confirming creation.

## Edit a rule

<ThemeImage lightSrc={rulesEditLight} darkSrc={rulesEditDark} alt="Editing an existing rule" />

Click the edit button to modify the domain, group name, or matching mode.

## Bulk actions

<ThemeImage lightSrc={rulesBulkActionsLight} darkSrc={rulesBulkActionsDark} alt="Multi-select with bulk action bar" />

Check multiple rules to enable, disable, or delete them in a single action.

## Import / Export

### Export rules

<ThemeImage lightSrc={rulesExportLight} darkSrc={rulesExportDark} alt="Export dialog with split button" />

Export all or selected rules as JSON. The split button lets you choose between copying to clipboard or downloading a file.

### Import rules

<ThemeImage lightSrc={rulesImportFileLight} darkSrc={rulesImportFileDark} alt="Import from file dialog" />

Import rules from a JSON file or by pasting text directly.

<ThemeImage lightSrc={rulesImportConflictsLight} darkSrc={rulesImportConflictsDark} alt="Conflict resolution during import" />

When conflicts occur (existing rules with the same domain), the wizard shows the differences and lets you choose which version to keep.
