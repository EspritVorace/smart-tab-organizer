## General settings

<ThemeImage lightSrc={settingsMiscLight} darkSrc={settingsMiscDark} alt="General settings page" />

The settings page groups the extension's behavior options:

- **Automatic grouping** — enables or disables tab grouping when tabs are opened
- **Deduplication** — configures duplicate tab detection mode (exact URL, path, domain, or substring)
- **Notifications** — enables notifications when grouping or deduplication occurs
- **Sync interval** — time between automatic syncs of pinned profiles
